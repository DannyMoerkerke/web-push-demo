Web Push Demo

Click "Subscribe" or "Unsubscribe" to subscribe to or unsubscribe from push notifications. You need to do this before you can receive push notifications.

When subscribed you can send a push notification. Enter a delay time in seconds to delay delivery of the notification.

Enter a long delay time so you have time to close the browser to test if the notification is also delivered when the browser is closed.

This works in Chrome on ChromeOS, Safari Tech Preview on MacOS Ventura and Chrome on Android.

Check the "Interaction" box to require the user to interact with the notification.

Normally, it will disappear after a few seconds but when this checkbox is checked it will only disappear after the user has interacted with it (either clicked or closed it). Supported only in Chrome.
